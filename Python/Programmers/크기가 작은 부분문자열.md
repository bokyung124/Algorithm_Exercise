## 2. 풀이

```python
def solution(t, p):
    answer = 0
    
    length = len(p)
    move = len(t) - len(p) +1
    
    for i in range(move):
        part = int(t[i:i+length])
        print(part)
        if part <= int(p):
            answer += 1
    
    return answer
```
